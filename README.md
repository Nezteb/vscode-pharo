## Features

We present here the existing features

### Generic feature

1. Code highlighting

### Supported Language Server feature

1. Code formatting
2. Hover
3. Auto-completion

### Additional feature

1. Save the Pharo image
2. Execute and show the result
3. Execute and print the result
4. Show the current server version
